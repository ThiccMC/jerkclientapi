cutieepc's jerkclient api

This is a custom API made to be compatible with a certain block game in combination with a certain client.

Currently available version: main branch retrieved on 15th June 2024, replay branch retrieved on 3rd July 2025.

Supported platforms include: Windows x64, Linux x64, macOS x64 and arm64. Linux arm64 support coming soon (on the main module only).

How to use?

Plug https://jerkapi.qtpc.tech as the API into the client. Launch it and the client will retrieve game files directly from this API.

It is highly recommended to delete your old game data before continuing.
